## Keyboard setting

basic: set [caps lock] to [control] at system preference

then use [Karabiner](https://pqrs.org/osx/karabiner/document.html.en)

implement:

- caps lock as L-ctrl

- single tap ctrl to esc

- ctrl + *anykey* : no rebind to esc.
